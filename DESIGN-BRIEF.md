## 依頼する画像一式

### 1. ヒーロー背景（最優先）
- **用途**：トップのスライダー背景（5枚）
- **サイズ**：2400 × 1200 px（横長、上下トリム耐性あり）
- **テーマ**：
  1. ピカピカの床（店舗 or オフィス）に反射する光
  2. 窓拭き中のスクイージーと水滴
  3. 高圧洗浄で外壁の汚れが落ちる瞬間
  4. 光る厨房・ステンレス
  5. 清掃プロの手元（プロ感・道具）
- **雰囲気**：暗めのネイビー基調 + ブルー/シアンの光、手前にスペース（文字が乗る）
- **人物**：顔は写さない、手元や後ろ姿のみ

### 2. サービスカード画像（6〜9枚）
- **サイズ**：800 × 600 px
- **種類**：
  - 床・窓・壁クリーニング
  - トイレ・カウンタークリーニング
  - 厨房・調理器具クリーニング
  - 排水管・排水溝高圧洗浄
  - 外壁・エントランス高圧洗浄
  - 特殊清掃（事故現場・ゴミ屋敷）
  - エアコン分解クリーニング
  - ワックス掛け
  - 定期清掃契約
- **スタイル**：before/after 感、ビジュアルに「清潔さの印象」

### 3. ロゴ
- **フォーマット**：SVG + PNG（透過）
- **バリエーション**：横型 / 縦型 / アイコン単体
- **モチーフ案**：水滴 + シールド / 円 / 光沢ライン
- **タイポ**：Barlow Condensed ベース、"Ai Clean Systems" 全社名＋"AI CLEAN" ショート版
- **サブコピー**：`Professional Cleaning Systems` など英文タグライン

### 4. 施工ギャラリー（before/after × 6組）
- **サイズ**：各 1200 × 800 px
- **組み合わせ**：トイレ / 厨房 / 床 / 排水管 / 外壁 / 特殊清掃

### 5. SNS / OGP 用
- **OGP**：1200 × 630 px（ロゴ + キャッチ）
- **プロフィールアイコン**：1024 × 1024 px（丸抜き対応）

### 6. バナー素材（任意）
- LINE 相談誘導バナー
- キャンペーンバナー（「今月限定◯◯円〜」等、差し替え可能なテンプレ化）

## 納品フォーマット
- **写真系**：WebP（第一希望）/ PNG
- **ロゴ系**：SVG + PNG 透過
- **配置**：`assets/images/` 配下に配置し、`index.html` の `data:image/...;base64,` 部分を `<img src="assets/images/xxx.webp">` に差し替え
- **Canva**：ブランドキットを作成し、今後の追加制作も同じトーンで量産できるようにしておく

## 参考：現状HTML内の仮画像箇所
`index.html` 内の `data:image/...;base64,` で始まる画像が差し替え対象。
grep で位置特定可能：
```bash
grep -n "data:image" index.html
```
